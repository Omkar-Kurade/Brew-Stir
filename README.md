# Brew-Stir Café Landing Page

A visually appealing and interactive landing page for a coffee and cocktail café. Built with **React**, **TailwindCSS**, and **GSAP**, this project showcases drinks, barista artistry, menu items, and contact information in a smooth, responsive, and animated design.

## Features

- **Responsive Design**: Works seamlessly across mobile, tablet, and desktop screens.
- **GSAP Animations**: Smooth scrolling and interactive animations for a modern experience.
- **TailwindCSS Styling**: Fully customized with reusable utility classes.
- **Interactive Menu Section**: Display cocktails and coffee options dynamically.
- **Hero Section**: Eye-catching introduction with a professional barista image.
- **Contact & Social Media Integration**: Easily link to Instagram, Facebook, and X (Twitter).
- **Favicon & Logo Integration**: Fully branded with a unique café logo.

## Technologies Used

- **React 19**
- **TailwindCSS 4**
- **GSAP (GreenSock Animation Platform)**
- **Vite** as the build tool
- **ESLint** for code quality and consistency
